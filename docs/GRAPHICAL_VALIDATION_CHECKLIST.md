# Graphical validation checklist

Run the client with `-Dorigins_overhaul.debug=true`, then execute `/originsoverhaul debug visual_lab`.

1. Select Phantom, rotate/zoom/pan, test `C`, `1`, and `F8`.
2. Select Feline and verify both ears and the six-segment tail on AUTO/CLASSIC/SLIM.
3. Select Merling and toggle `2` for underwater; verify fins and bubbles.
4. Select Enderian and inspect emissive eyes under dark lighting.
5. Run `/originsoverhaul debug visual_world` and compare the real player profile.
6. Repeat with outer skin layers, head/chest armor, Sodium and Iris where available.

The agent environment is headless, so this checklist must be completed by a user with a real Minecraft window before graphical validation is marked complete.
